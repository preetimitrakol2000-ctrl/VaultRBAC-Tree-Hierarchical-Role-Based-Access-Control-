# VaultRBAC-Tree // Hierarchical Role Access Controller

A zero-trust identity system analyzing corporate access constraints using performant C memory structures and a local policy contextual expansion RAG loop.

## Architecture
- **N-ary Inheritance Map (C):** Models enterprise identity charts natively on the heap to execute path validation in recursive trees.
- **Policy RAG System (Python):** Computes cosine distances across failing authorization codes to output system permission overrides.

## Execution
```bash
python policy_rag.py
